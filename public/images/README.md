# Folder Images Portfolio

Silakan tempatkan foto-foto Anda di folder ini dengan nama file berikut:

## File Foto yang Dibutuhkan:

1. **profile.jpg** - Foto profil untuk sidebar (ukuran disarankan: 300x300px atau lebih besar, bentuk persegi)
   - Akan ditampilkan di sidebar kiri dalam bentuk lingkaran
   - Disarankan foto close-up atau portrait

2. **main-image.jpg** - Foto utama untuk halaman beranda (ukuran disarankan: 600x800px atau lebih besar, bentuk portrait)
   - Akan ditampilkan di halaman utama (home page)
   - Bisa menggunakan foto artistik dengan efek atau foto profesional

## Format File yang Didukung:
- JPG/JPEG
- PNG
- WebP

## Tips:
- Gunakan foto berkualitas tinggi untuk hasil terbaik
- Pastikan foto tidak terlalu besar (maksimal 2MB per file) untuk performa website yang optimal
- Untuk foto profil, pastikan subjek berada di tengah frame
- Untuk foto utama, foto portrait (tegak) akan terlihat lebih baik
